## Описание:

```
tasks.c — реализация распараллеливания с помощью #pragma omp task

for.c — реализация распараллеливания с помощью #pragma omp for

2mm_new.c — оптимизация исходной программы, без распараллеливания

2mm_og.c — исходная (неоптимизированная) программа

run*.sh - скрипты для проведения эксперимента по множественному компиляции\запуску программа
с разными параметрами

run*.txt - результаты тестовых(демо) проверок

res_@dataset_name.txt - результаты запуска программы на сервере К10
```
